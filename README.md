# 📘 DSA Programs (C++)

A collection of **Data Structures and Algorithms (DSA)** programs implemented in **C++**, covering fundamental concepts and commonly used algorithms.

---

## 🚀 About This Repository

This repository contains basic to intermediate DSA programs written in C++ for learning and practice. Each file demonstrates a specific concept or algorithm, making it useful for:

* 📚 Academic learning
* 💼 Interview preparation
* 🧠 Strengthening problem-solving skills

---

## 📂 Project Structure

### 🔢 Array & Matrix

* `array.cpp` – Basic array operations
* `matrix.cpp` – Matrix operations

### 🔍 Searching

* `binarysearch.cpp` – Binary Search implementation

### 📊 Sorting Algorithms

* `bubble.cpp` – Bubble Sort
* `insertion.cpp` – Insertion Sort
* `selection.cpp` – Selection Sort

### 🌐 Graph Algorithms

* `bfs.cpp` – Breadth First Search
* `dfs.cpp` – Depth First Search

### 🔗 Data Structures

* `linkedlist.cpp` – Linked List implementation
* `enque.cpp` – Queue operations
* `pushpop.cpp` – Stack operations

### 🔁 Recursion / Misc

* `1d.cpp` – 1D array example
* `2d.cpp` – 2D array example
* `order_run.cpp` – Execution order logic

### ⚙️ Other Files

* `crc.cpp` – Cyclic Redundancy Check
* `a.exe` – Compiled output file (can be ignored)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Riteshlande75/DSA-programs.git
   ```

2. Navigate to the folder:

   ```bash
   cd DSA-programs
   ```

3. Compile any program:

   ```bash
   g++ filename.cpp -o output
   ```

4. Run the program:

   ```bash
   ./output
   ```
